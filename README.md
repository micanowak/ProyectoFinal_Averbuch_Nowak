## Getting Started

Para Iniciar el proyecto y abrirlo correctamente, los siguientes aspectos serán importantes.

### Prerequisites

- Tener instalado Visual Studio Code

### Installation

1. Clonar el repositorio
   ```
   git clone https://github.com/micanowak/ProyectoFinal_Averbuch_Nowak.git
   ```
2. Dividir la terminal para instalar las distintas dependencias en cada carpeta. 
3. En una ventana deberás abrir la carpeta 'Api' y en la otra 'MobileApp'
    ```
    cd ./Api/
    ```
    ```
    cd ./MyApp/
    ```
4. En cada ventana por separado realizar el comando:
    ```
    npm i
    ```
5. En la ventana de 'ProyectoFinal_Averbuch_Nowak\MobileApp' instalar 'Expo' con el siguiente comando:
    ```
    npm i expo-cli -g
    ```
6. Abrir MSSQL y pegar en una nueva query la base de datos.
7. En 'ProyectoFinal_Averbuch_Nowak\Api\.env' cambiar 'DB_SERVER' por el servidor en uso.
8. Copiar lo que esta comentado en 'ProyectoFinal_Averbuch_Nowak\Api\dbconfig.js' y pegarlo en una nueva query

<!-- USAGE EXAMPLES -->
## Usage

Para ser utilizado, seguir los siguientes pasos:
1. Dividir en dos una nueva Terminal
2. En una ventana deberás abrir la carpeta 'Api' y en la otra 'MyApp'
    ```
    cd ./Api/
    ```
    ```
    cd ./MyApp/
    ```
3. En la ventana de 'Api':
    ```
    node index.js
    ```
4. En la ventana de 'MyApp':
    ```
    expo start --tunnel
    ```
